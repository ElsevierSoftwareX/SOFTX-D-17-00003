Checks whether a sequence is self-complementary, so the complement of all tuples is part of the same sequence.

The complement of a tuple is created by replacing the following nucleoabases and reversing the resulting tuple:
AdeninewithUracil
UracilwithAdenine
GuaninewithCytosine
CytosinewithGuanine

Example:
Input sequence:AGU UGA UCA ACU UCG CGA
Test result:Pass

Example:
Input sequence:AGU UGA UAC GCA CAG
Test result:Fail
