Splits a sequence based on a binary dichotomic algorithms (BDA). All tuples are moved to one class of the binary dichotomic alogrithm.

Use the BDA editor define a set of binary dichotomic algorithms, the number of classes are displayed in the lower right of the editor.

Example Rumer:
Input sequence:CAC UGC AUA ACG AGC GCU
Output sequences:ACG GCU
CAC UGC AUA AGC
