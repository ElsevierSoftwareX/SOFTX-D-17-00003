The Genetic Code Analysis Toolkit (GCAT) features a comprehensive editor to edit and analyze genetic code sequences and sets.

The user interface (UI) is separated into five areas with different purpose:

  • The input area on the left side can be used to quickly add codons to the editor.
  • Underneath the input area, you'll find all available operations. Operations are separated into different categories:
    • Test: Tests a genetic sequence for a specific question and returns a boolean result.
    • Transformation: Transforms a given genetic sequence into another sequence based on certain rules.
    • Split: Splits a sequence into one or more sub-sequences.
    • Analysis: Performs an arbitrary analysis on a genetic sequence.
  • The editor is the input area of the tool. By typing you can add nucleotides to the editor.
  • On the right side of the editor is the display area. Different kinds of displays feature a graphical view on the sequence or provide additional information.
  • Below the editor is the output area, which displays the result of tests, analysis or other operations.

On top of the tool, you will find access to all operations in the menu- and toolbar.

Open / Import File

The Toolkit features an own file format (.gen) to store and load files containing genetic code sequences. Using the import functionality you can open FATSA files (.fasta) or import genetic sequences from GenBank.
