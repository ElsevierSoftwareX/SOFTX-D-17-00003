The Genetic Code Analysis Toolkit (GCAT) features a comprehensive editor to edit and analyze genetic code sequences and sets (codes).

The user interface (UI) is separated into five areas with different purposes:

  • The input area on the left side can be used for quick adding of codons to the editor.
  • Underneath the input area, all available operations. Operations are separated into different categories:
    • Test: Tests a genetic sequence or code for a property specified and returns a boolean result.
    • Transformation: Transforms a given genetic sequence or code into another sequence or code based on rules specified.
    • Split: Splits a sequence or a code into sub-sequences or sub-codes.
    • Analysis: Performs an analysis on a specified genetic sequence.
  • The editor is the input area of the tool.
  • On the right side of the editor is the display area. Different kinds of displays feature a graphical view on the sequence or provide additional information.
  • Below the editor is the output area, which displays the result of tests, analysis or other operations.

On top of the tool, all operations in the menu- and toolbar are accessible.

Open / Import File

The Toolkit features an own file format (.gen) to store and load files containing genetic code sequences. Using the import functionality you can open FASTA files (.fasta) or import genetic sequences from GenBank.
