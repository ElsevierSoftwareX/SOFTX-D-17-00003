Performs one of the following common substitutions to the sequence. All nucleobases in the sequence will be subtituted based on the following tables:

id Transformation:
AdeninewithAdenine
CytosinewithCytosine
GuaninewithGuanine
UracilwithUracil

c Transformation:
AdeninewithUracil
CytosinewithGuanine
GuaninewithCytosine
UracilwithAdenine

p Transformation:
AdeninewithGuanine
CytosinewithUracil
GuaninewithAdenine
UracilwithCytosine

r Transformation:
AdeninewithCytosine
CytosinewithAdenine
GuaninewithUracil
UracilwithGuanine

Π Transformations: Swap the specified nucleobases for all given tuples.

Example c Transformation:
Input sequence:AGU UGA UAC GCA CAG
Output sequence:UCA ACU AUG CGU GUC
