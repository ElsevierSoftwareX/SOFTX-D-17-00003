Calculates the run length for a given C3 code. A run is a continuous sequence of tupels that belong to a C3 code which is embedded by runs of tupels that do not belong to a C3 code or vice versa. The output is 1) the minimum run length, 2) the average run length and 3) the maximum run length for C3 and non C3 runs.

Parameter
1st:Number of the C3 set, i.e. a number between 1 and 216.

Example for C3(1):
Input sequence:AUU AAC GCA GCC GCU UCU UAC CAG UUG ACA UCA AAC UAA AGA CGG UAU GAG GAG ACC UGG ACC CAA AUG CGA GCC CGC GGG AUC CAC GAU CUU ACG CGG GUA ACC CGU AGA ACC UCA AGG
Analysis result:
    C3(1): min = 1, avg = 1.78, max = 3
not C3(1): min = 1, avg = 3.00, max = 9
