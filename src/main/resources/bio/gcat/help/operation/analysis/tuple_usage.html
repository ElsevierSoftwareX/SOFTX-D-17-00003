Analyzes a given file, for how often a tuple has been used in the file.

All tuples of the given sequence are counted in the file, the result is a list of frequencies of all tuples of the sequence.

Example:
Input sequence:ACG CUC CAC UCA AGU
Analysis result:59x ACG, 72x CUC, 72x CAC, 98x UCA, 98x AGU
