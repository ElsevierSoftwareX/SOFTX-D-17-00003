Graphically visualizes, which tuples (codons) of a given sequence are part of a dichotomic classes of a binary dichotomic algorithm. Currently, only codons (i.e. tuples of length 3) are allowed.

Parameter
The BDA set as defined in the BDA Tool is taken.

Example for Rumer partition:
Input sequence:ACG CUC UGU UCG UUU GCG
Analysis result:ACG CUC UGU UCG UUU GCG
