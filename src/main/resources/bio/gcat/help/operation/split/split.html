The "Split" functionality allows you to split a given sequence at a certain delimiter.

The delimiter specifies where the sequence should be split. The delimiter is located using the Find functionality. Afterwards the sequence is split at the found delimiter positions and the result is displayed.

Splitting a sequence "AGU ACA AGG" with the delimiter "ACA" will result in two sequences: "AGU ACA" and "AGG". Choosing "Remove delimiter" will remove the delimiter on the split, so the result of the previous example would be "AGU" and "AGG".

Example: Split at "UGC"
Input sequence:CAC UGC CAG UGC AUA UGC
Output sequences:UGC
CAC UGC
CAG UGC
AUA UGC
