Very similar to the Split functionality, "Pick" will search the sequence for given tuples and pick all matching ones.

The result of this operation will always be two sequences. One sequence containing all matching tuples and one sequence containing the remaining tuples of the sequence.

Example: Pick ".GC"
Input sequence:CAC CGC CAG UGC AUA AGC
Output sequences:UGC
CGC UGC AGC
CAC CAG AUA
